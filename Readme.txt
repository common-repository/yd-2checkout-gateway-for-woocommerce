YD Gateway 2Checkout for WooCommerce provides an easy way to take credit card payments on your online store using 2Checkout.

== Description ==
YD Gateway 2Checkout for WooCommerce is a quick and easy solution for taking credit card and PayPal payments on your online store using 2Checkout. The plugin displays payments without visiting 2Checkout site via a popup window on your checkout page. To register and setup your 2Checkout account, visit [this page](https://www.2checkout.com/referral?r=alidropshipx2co).
YD Gateway 2Checkout for WooCommerce is fully compatible with WordPress and WooCommerce latest versions.
If you find something wrong in plugin's functioning, I am always open for discussion and glad to hear any suggestions, questions and notes.

**Features:**
- Simple setup
- A popup window on your checkout page
- Enable/Disable demo mode
- Includes PayPal payments
- Compatible with WooCommerce

== Installation ==
To install the extension, please follow this instruction.
1. Upload `yd-2checkout-gateway` folder to the `/wp-content/plugins/` directory, or install the extension through the WordPress plugins screen directly.
2. Activate the extension through the 'Plugins' menu in WordPress.
3. The settings will be available in WooCommerce - Settings - Payments.

== Frequently Asked Questions ==
= How to setup 2Checkout account? =
To register and setup your 2Checkout account, visit [this page](https://www.2checkout.com/referral?r=alidropshipx2co).
= How can I find this option in my WP admin area?
Open WooCommerce - Settings - Payments, select 2Checkout in popup with Credit Card/PayPal and click Manage.
